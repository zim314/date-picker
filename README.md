# README

## 安裝

以下將會引導你如何安裝此專案到你的電腦上。
Node.js 版本建議為：`18.18.0` 以上。

<br/>

### 取得專案

```bash
git clone https://github.com/zim314/date-picker.git
```

<br/>

### 移動到專案內

```bash
cd date-picker
```

<br/>

### 安裝套件

```bash
npm i
```

<br/>


### 運行專案

```bash
npm run dev
```

<br/>

### 開啟專案

在瀏覽器網址列輸入以下即可看到畫面

```bash
http://localhost:5173/date-picker
```
